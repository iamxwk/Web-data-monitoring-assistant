请输入JSON格式的请求头，例如: {"Content-Type": "application/json", "Authorization": "Bearer token"}
请输入请求体内容，对于POST/PUT等请求方法有效

请输入JavaScript代码，必须返回包含content和notify字段的对象。

content：字符串类型，用于比较数据变化的值

notify：布尔值类型，true 表示显示提醒

extra：对象类型，额外信息，可在后续处理中使用

Return example： 
return {
  "content": taskData.content,
  "notify": taskData.content != taskData.prevContent,
  "extra": {}
};
